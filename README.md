# PS-handle-lot-pics-in-your-database
###中文版本介绍###

这个是一个Javascript的PS的脚本用于批量处理文件树状结构下的所有子文件夹内有图片的文件（结合ps独有的动作功能，您可以实现各种各样的ps自动化流程）我在这个项目使用的过程当中处理了几百万张的基础图片数据(基本就是修改亮度或者图片的水印)结合PS的超级强大的图像算法，可以做到非常好玩的ps自动化流程。

本项目是基于python及其javascript的语言完成PS的自动化的处理

未来这个项目将会基于图像处理算法独立出不依赖PS的版本；专注于大量数据库的图像批量处理



###EN Introduce###

This is a JavaScript PS script used for batch processing all subfolders containing image files under a file tree structure (combined with PS's unique action features, you can achieve various PS automation workflows). In the process of using this project, I processed millions of basic image data (mostly just adjusting brightness or adding watermarks). Combined with PS's super powerful image algorithms, it is possible to create very interesting PS automation workflows.

This project is built using Python and JavaScript to achieve PS automation processing.

In the future, this project will be based on image processing algorithms to develop a PS-independent version, focusing on large-scale batch processing of database images.
